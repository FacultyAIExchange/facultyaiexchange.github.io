# Add an item

To add a gallery item, create a new Markdown file in `_apps`. Start from `docs/gallery-item-template.md`, keep the front matter fields, and write the details in ordinary Markdown below the front matter.

Each item should explain:

- what faculty used the AI application for;
- what made it useful in practice;
- what materials another person would need to adapt it;
- which skill files, prompts, or examples are relevant.

Related skill files can be stored under `skills/your-skill-name/SKILL.md` and linked from the item's `skill_files` list.
