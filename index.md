# Faculty AI Gallery

This gallery collects practical examples of generative AI applications that faculty have found useful. Each entry is meant to be specific enough that another instructor, researcher, or staff partner can understand the problem being solved, the workflow that helped, and the reusable skill files or prompts that supported the work.

The first version is intentionally lightweight. Contributors can add a Markdown file to the `_apps` folder, include a short summary and metadata in the front matter, and link to any related skill files stored in the repository.
